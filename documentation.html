Overview

This website is designed for anime enthusiasts, providing a unique platform to explore various anime series. The website offers the following features:

Anime Quotes

This feature displays random quotes from various anime series. It uses the AnimeChan API to fetch the quotes.

Usage

Simply visit the Anime Quotes page on our website. A random quote from an anime series will be displayed. Refresh the page to view a new quote.

Anime Rankings

This feature provides up-to-date rankings of anime series. It uses the MyAnimeList API to fetch the rankings.

Usage

Visit the Anime Rankings page on our website. The page displays a list of anime series ranked according to their ratings.

Voice Commands

This feature allows users to navigate the website using voice commands. It uses the Annyang library to implement this feature.

Usage

To use voice commands, click on the microphone icon on the website. Once activated, you can use voice commands to navigate through the website, search for anime, and much more.

Anime Background Citation

Background to fetch anime backgrounds.

Supabase Table

Supabase was used to fetch table data and uses the contact form to obtain data from users and input it into the supabase table.

Conclusion

This website aims to provide a unique and interactive platform for anime enthusiasts. We hope you enjoy using it!
